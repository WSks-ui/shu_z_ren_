# 研伴 - 智能教育数字人系统

## 概述

研伴是基于 super-agent-party 开源项目开发的**智能教育数字人 Web 应用**，旨在为研究者和学生提供全方位的学术研究辅助。系统集成了 3D 数字人形象、五大专业教育技能、成长激励系统、人机协作记录等核心功能，实现有温度的智能伴学体验。

**项目用途**：普通高等教育组竞赛演示

---

## 核心功能

### 1. 3D 教育数字人

系统采用**魔珐星云**引擎驱动的 3D 数字人形象，提供有温度的交互体验：

| 特性 | 说明 |
|------|------|
| **云端渲染** | 高质量 3D 数字人形象，无需本地 GPU |
| **语音驱动** | 支持文本驱动数字人说话，唇形同步 |
| **情绪联动** | 根据对话内容动态调整数字人表情 |
| **实时交互** | 低延迟响应，流畅的对话体验 |

- **网关地址**：`https://nebula-agent.xingyun3d.com/user/v1/ttsa/session`
- 支持连接/断开控制
- 支持快速文本播报

### 2. 五大专业教育技能

系统提供五种专业教育技能，每种技能采用**苏格拉底式引导**方法，通过提问帮助用户独立思考：

| 技能 | 图标 | 核心能力 | 对话阶段 |
|------|------|----------|----------|
| **科研助手** | fa-flask | 实验设计、研究方法、数据分析指导 | 选题探索 → 假设构建 → 实验设计 → 数据分析 |
| **文献综述** | fa-book-open | PICO 框架、PRISMA 流程、检索策略 | 主题界定 → 检索策略 → 筛选评估 → 综合撰写 |
| **论文写作** | fa-pen-fancy | 结构规划、写作技巧、多格式引用 | 论文选题 → 框架搭建 → 内容撰写 → 润色定稿 |
| **虚拟导师** | fa-graduation-cap | 知识讲解、学习规划、答疑解惑 | 学情诊断 → 知识讲解 → 练习巩固 → 总结提升 |
| **数学助手** | fa-square-root-variable | 公式识别、分步解题、错题分析 | 问题理解 → 方法选择 → 逐步求解 → 验证总结 |

**技能通用特性**：
- 支持图片识别（公式、文献、笔记、图表等）
- 支持阶段跳转指令 `[跳转阶段:阶段名]`
- 支持番茄钟时间管理 `[设置番茄:25分钟][开始番茄]`
- 支持笔记生成 `[生成笔记]`
- 支持多轮对话和长期记忆
- 每个技能独立追踪使用次数

### 3. 成长与成就系统

#### 经验与等级
| 功能 | 说明 |
|------|------|
| **经验值** | 通过对话、使用技能、解锁成就获得经验值 |
| **等级系统** | 每 100 经验升一级，支持多级成长 |
| **技能统计** | 分别统计各技能使用次数 |
| **综合统计** | 对话次数、论文阅读、实验设计、学习时长等 |

#### 成就徽章系统

成就分为四大类别，共 12 个徽章：

| 分类 | 成就 | 解锁条件 | 奖励经验 |
|------|------|----------|----------|
| **基础成就** | 初次对话 | 完成第一次对话 | +30 |
| | 技能探索者 | 使用所有技能各1次 | +50 |
| **进阶成就** | 进阶学者 | 达到5级 | +100 |
| | 协作大师 | 完成20次协作会话 | +80 |
| | 番茄大师 | 完成10个番茄钟 | +60 |
| | 知识收藏家 | 收藏20条知识书签 | +70 |
| **挑战成就** | 资深学者 | 达到10级 | +200 |
| | 任务达人 | 完成50个待办任务 | +150 |
| | 公式达人 | 识别10个手写公式 | +100 |
| | 阶段大师 | 完成20个学习阶段 | +120 |
| **隐藏成就** | 夜间学者 | 在深夜(23:00-5:00)使用研伴 | +50 |
| | 语音先锋 | 进行10次语音对话 | +80 |

**成就系统特性**：
- 点击徽章查看详细信息（描述、分类、奖励、解锁时间）
- 隐藏成就未解锁时显示神秘问号图标
- 解锁时自动发放经验奖励
- 成就进度实时追踪

### 4. 人机协作记录

- **协作类型**：论文协作、实验协作、综述协作、辅导协作
- **贡献追踪**：分别记录 AI 和用户的贡献内容
- **协作比例**：自动计算人机协作贡献比例
- **报告导出**：支持导出 Word 格式学习报告
- **选择性导出**：支持按类型筛选、指定会话导出

#### 学习报告功能

学习报告包含以下内容：

| 模块 | 内容 |
|------|------|
| **核心统计** | 学习会话数、总贡献、对话次数、当前等级 |
| **成就时间线** | 最近解锁的成就及解锁日期 |
| **技能使用趋势** | 各技能使用次数条形图、最常用技能标注 |
| **智能学习建议** | 优势领域、改进建议、下一步目标、学习技巧 |

### 5. 情绪映射引擎

- **情绪分析**：支持 10+ 种情绪识别（开心、困惑、好奇、沮丧等）
- **表情联动**：数字人根据情绪状态展现不同表情
- **历史记录**：记录对话过程中的情绪变化
- **玻璃态 UI**：实时显示当前情绪状态

### 6. 数学公式支持

- **手写识别**：集成 SimpleTex API，识别手写公式
- **KaTeX 渲染**：实时渲染 LaTeX 数学公式
- **分步解答**：每一步标注数学依据
- **多格式支持**：支持代数、微积分、线性代数、概率统计等

### 7. 智能笔记系统

- **AI 生成笔记**：对话中输入 `[生成笔记]` 自动从对话生成笔记
- **Markdown 编辑**：支持富文本编辑和预览
- **Word 导出**：支持导出为 Word 文档
- **笔记管理**：按技能分类、搜索、删除

### 8. 番茄钟时间管理

- **AI 触发**：对话中输入 `[设置番茄:25分钟][开始番茄]` 自动启动
- **计时功能**：工作时间/休息时间自动切换
- **任务关联**：可关联待办任务
- **背景音乐**：可选专注音乐
- **统计追踪**：完成番茄钟计入成长统计

### 9. 教育知识库

基于 RAG 技术的向量化知识库，包含：

| 文档 | 内容 |
|------|------|
| 学术写作规范 | 论文结构、语言风格、引用格式指南 |
| 教育研究方法 | 定量/定性研究设计、数据分析方法 |
| 跨学科研究方法 | 跨学科研究范式、整合策略 |
| AI 局限性说明 | AI 能力边界、学术伦理规范 |

### 10. 学术 API 集成

| API | 用途 | 说明 |
|-----|------|------|
| arXiv | 论文检索 | 计算机科学、物理等领域预印本 |
| Semantic Scholar | 学术搜索 | 2亿+ 论文，引用分析 |
| CrossRef | 文献查询 | 最大 DOI 注册机构 |
| OpenAlex | 开放学图谱 | 2.5亿+ 文献 |

## 常见问题

### 1. Python 虚拟环境未找到

**错误信息**: `spawn .venv\Scripts\python.exe ENOENT`

**原因**: 未创建 Python 虚拟环境

**解决方案**:
```bash
# 创建虚拟环境
python -m venv .venv

# 安装 Python 依赖
# Windows:
.venv\Scripts\pip.exe install -r requirements.txt
# Linux/Mac:
.venv/bin/pip install -r requirements.txt
```

### 2. SSL 证书问题导致健康检查超时

**错误信息**: `后端已启动但健康检查响应超时` 或 `Invalid HTTP request received`

**原因**: 后端需要 HTTPS，但缺少 SSL 证书文件

**解决方案**:
```bash
# 创建证书目录
mkdir certs

# 生成自签名 SSL 证书（需要安装 openssl）
cd certs
openssl req -x509 -newkey rsa:4096 -keyout key.pem -out cert.pem -days 365 -nodes -subj "//CN=localhost"
```

或者使用环境变量指定证书路径：
```bash
# Windows CMD
set SSL_KEYFILE=D:\path\to\key.pem
set SSL_CERTFILE=D:\path\to\cert.pem

# Windows PowerShell
$env:SSL_KEYFILE="D:\path\to\key.pem"
$env:SSL_CERTFILE="D:\path\to\cert.pem"

# Linux/Mac
export SSL_KEYFILE=/path/to/key.pem
export SSL_CERTFILE=/path/to/cert.pem
```

### 3. Electron 下载超时（国内网络）

**错误信息**: `RequestError: connect ETIMEDOUT` 或 `npm install` 卡住

**原因**: Electron 二进制文件下载超时

**解决方案**:
```bash
# 使用国内镜像
# Windows CMD
set ELECTRON_MIRROR=https://npmmirror.com/mirrors/electron/
npm install

# Windows PowerShell
$env:ELECTRON_MIRROR="https://npmmirror.com/mirrors/electron/"
npm install

# Linux/Mac
ELECTRON_MIRROR=https://npmmirror.com/mirrors/electron/ npm install
```

### 4. node_modules 目录被锁定

**错误信息**: `EBUSY: resource busy or locked`

**原因**: 有进程占用了 node_modules 目录

**解决方案**:
1. 关闭所有可能占用的程序（VS Code、其他终端、资源管理器等）
2. 重新打开终端，删除 node_modules 后重新安装：
```bash
# Windows
rmdir /s /q node_modules
npm install

# Linux/Mac
rm -rf node_modules
npm install
```

### 5. 端口被占用

**错误信息**: `Address already in use` 或 `端口 3456 被占用`

**解决方案**:
```bash
# Windows - 查找并结束占用端口的进程
netstat -ano | findstr :3456
taskkill /PID <进程ID> /F

# Linux/Mac
lsof -i :3456
kill -9 <进程ID>
```

5. 查看各项统计数据的详细信息

### 管理协作记录

1. 点击「协作记录」标签页
2. 使用类型筛选查看特定协作
3. 点击「学习报告」查看：
   - 成就解锁时间线
   - 技能使用趋势图
   - 智能学习建议
4. 导出 Word 格式学习报告

### 使用番茄钟

1. 点击快捷栏番茄钟按钮
2. 或在对话中说"帮我设置一个25分钟的番茄钟"
3. 开始专注学习
4. 完成后获得经验奖励

### 数学公式使用

1. 在数学助手模式下
2. 使用手写板输入公式
3. 系统自动识别并渲染
4. 获取详细解题步骤

## 学术伦理声明

本系统明确了 AI 技术的局限性，并提供学术伦理指导：

### AI 适合辅助的场景
- 文献检索与整理

- 实验设计决策
- 数据分析与解释
