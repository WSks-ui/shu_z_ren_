# 教育数字人系统

## 概述

本系统是基于 super-agent-party 项目开发的教育数字人 Web 应用，旨在支持普通高等教育组竞赛。系统集成了 VRM 3D 数字人、腾讯智能数智人、教育技能、成长系统和人机协作记录等功能。

## 核心功能

### 1. 双数字人引擎

系统支持两种数字人引擎：

| 引擎 | 特点 | 适用场景 |
|------|------|----------|
| **VRM 数字人** | 本地渲染、开源免费、可定制 | 开发测试、本地部署 |
| **腾讯数智人** | 云端渲染、高质量形象、TTS集成 | 生产环境、商业演示 |

### 2. 教育技能

系统提供四种专业教育技能：

| 技能 | 说明 |
|------|------|
| **科研助手** | 苏格拉底式引导实验设计，文献检索支持 |
| **文献综述** | PRISMA 系统综述方法指导 |
| **论文写作** | 学术论文写作协助 |
| **虚拟导师** | 个性化学习支持和答疑 |

### 3. 成长系统

- **经验值系统**：通过协作活动获得经验值
- **等级系统**：每 100 经验升一级
- **成就系统**：完成特定目标解锁成就
- **统计追踪**：对话次数、论文阅读、实验设计等

### 4. 人机协作记录

- 记录 AI 和人类用户在协作中的贡献
- 区分不同类型的协作：论文、实验、综述、辅导
- 统计分析协作比例

### 5. 情绪映射引擎

- 文本情绪分析（支持 10+ 种情绪）
- 数字人表情联动（VRM 和腾讯数智人）
- 情绪历史记录

## API 端点

### 教育系统 API (`/api/education`)

| 端点 | 方法 | 说明 |
|------|------|------|
| `/growth` | GET/POST | 获取/保存成长数据 |
| `/growth/add_exp` | POST | 增加经验值 |
| `/achievements` | GET | 获取成就列表 |
| `/achievements/unlock` | POST | 解锁成就 |
| `/collaboration` | GET/POST | 获取/保存协作记录 |
| `/collaboration/start_session` | POST | 开始协作会话 |
| `/collaboration/add_contribution` | POST | 添加贡献记录 |
| `/collaboration/end_session` | POST | 结束协作会话 |
| `/emotion/record` | POST | 记录情绪状态 |
| `/skills/record_usage` | POST | 记录技能使用 |

### 腾讯数智人 API (`/api/tencent_digital_human`)

| 端点 | 方法 | 说明 |
|------|------|------|
| `/create_session` | POST | 创建数字人会话 |
| `/close_session` | POST | 关闭数字人会话 |
| `/start_session` | POST | 开始数字人会话 |
| `/stat_session/{id}` | GET | 查询会话状态 |
| `/drive_text` | POST | 文本驱动数字人 |
| `/drive_audio` | POST | 音频驱动数字人 |
| `/drive_emotion` | POST | 情绪驱动数字人 |
| `/connect_websocket` | POST | 连接 WebSocket 驱动通道 |
| `/active_sessions` | GET | 获取活跃会话列表 |
| `/health` | GET | 健康检查 |

### VRM 控制端点

| 端点 | 方法 | 说明 |
|------|------|------|
| `/vrm_emotion` | POST | 设置 VRM 表情 |
| `/vrm_config` | GET | 获取 VRM 配置 |

## 使用方法

### 1. 启动服务

```bash
python server.py
```

### 2. 访问界面

打开浏览器访问 `http://localhost:3456`

### 3. 使用教育功能

1. 在左侧菜单点击「教育数字人」
2. 选择数字人类型（VRM 或腾讯数智人）
3. 如选择腾讯数智人，点击「连接」启动会话
4. 选择一个教育技能模式
5. 开始对话，系统会自动记录协作过程
6. 查看成长面板了解进度

### 4. 腾讯数智人配置

系统已预配置腾讯数智人 API 密钥。如需修改，编辑 `py/tencent_digital_human.py` 中的 `TENCENT_CONFIG`：

```python
TENCENT_CONFIG = {
    "appkey": "your_appkey",
    "accesstoken": "your_accesstoken",
    "virtualmanProjectId": "your_project_id",
    "sign": "your_sign",
}
```
