# 教育数字人系统 - 使用指南

## 快速启动

### 方式一：双击启动脚本
双击 `启动教育数字人.bat` 文件，系统会自动启动服务并打开独立界面。

### 方式二：手动启动
```bash
# 1. 启动后端服务
py server.py --host 127.0.0.1 --port 3456

# 2. 打开浏览器访问
# 独立界面: http://127.0.0.1:3456/education-digital-human.html
# 主界面:   http://127.0.0.1:3456
```

## 访问地址

| 界面 | 地址 | 说明 |
|------|------|------|
| **独立界面** | http://127.0.0.1:3456/education-digital-human.html | 专为教育数字人设计的独立界面 |
| 主界面 | http://127.0.0.1:3456 | 完整的 super-agent-party 界面 |

## 独立界面功能

### 左侧面板 - 数字人
- **VRM 数字人**: 本地渲染的 3D 数字人
- **腾讯数智人**: 云端渲染的高质量数字人
  - 点击「连接」启动会话
  - 输入文字让数字人播报

### 右侧面板 - 功能区

#### 1. 教育技能
- **科研助手**: 苏格拉底式引导实验设计
- **文献综述**: PRISMA 系统综述方法
- **论文写作**: 学术论文写作指导
- **虚拟导师**: 个性化学习支持

#### 2. 成长系统
- 等级和经验值追踪
- 学习统计数据
- 成就徽章系统

#### 3. 协作记录
- 人机协作历史
- AI 和用户贡献统计

#### 4. 对话
- 与教育数字人实时对话
- 支持 Markdown 格式

## 腾讯数智人配置

系统已预配置 API 密钥。如需修改，编辑 `py/tencent_digital_human.py`:

```python
TENCENT_CONFIG = {
    "appkey": "your_appkey",
    "accesstoken": "your_accesstoken",
    "virtualmanProjectId": "your_project_id",
    "sign": "your_sign",
}
```

## 参赛使用

### 演示流程
1. 双击 `启动教育数字人.bat`
2. 选择「腾讯数智人」标签
3. 点击「连接」启动数字人
4. 选择一个教育技能
5. 点击「开始使用」进入对话
6. 展示人机协作和成长系统

## 常见问题

### Q: 腾讯数智人连接失败？
A: 检查网络连接，确保可以访问 `gw.tvs.qq.com`

### Q: VRM 数字人不显示？
A: 确保浏览器支持 WebGL，或尝试刷新页面

### Q: 对话无响应？
A: 检查后端服务是否正常运行，查看控制台日志
