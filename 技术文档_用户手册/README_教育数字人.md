# 教育数字人系统

## 概述

本系统是基于 super-agent-party 项目开发的教育数字人 Web 应用，旨在支持普通高等教育组竞赛。系统集成了魔珐星云 3D 数字人、教育技能、成长系统和人机协作记录等功能。

## 核心功能

### 1. 魔珐星云数字人

系统采用魔珐星云(XingYun)数字人引擎：

| 特性 | 说明 |
|------|------|
| **云端渲染** | 高质量3D数字人形象，无需本地GPU |
| **TTS集成** | 文本转语音驱动数字人说话 |
| **实时交互** | 低延迟响应，流畅的对话体验 |
| **API控制** | RESTful API接口，易于集成 |

网关地址：`https://nebula-agent.xingyun3d.com/user/v1/ttsa/session`

### 2. 教育技能

系统提供五种专业教育技能：

| 技能 | 图标 | 说明 |
|------|------|------|
| **科研助手** | fa-flask | 苏格拉底式引导实验设计，培养科研思维 |
| **文献综述** | fa-book-open | PRISMA系统综述方法，多数据库协同检索 |
| **论文写作** | fa-pen-fancy | 学术论文写作指导，支持多种引用格式 |
| **虚拟导师** | fa-graduation-cap | 个性化学习支持，答疑解惑 |
| **数学助手** | fa-square-root-variable | 手写公式识别，数学推导与解题 |

每种技能支持4阶段成长体系：入门 → 进阶 → 熟练 → 专家

### 3. 成长系统

- **经验值系统**：通过协作活动获得经验值
- **等级系统**：每 100 经验升一级
- **成就系统**：完成特定目标解锁成就
- **统计追踪**：对话次数、论文阅读、实验设计等
- **技能成长**：每个教育技能独立追踪成长进度

### 4. 人机协作记录

- 记录 AI 和人类用户在协作中的贡献
- 区分不同类型的协作：论文、实验、综述、辅导
- 统计分析协作比例
- 支持协作会话管理

### 5. 情绪映射引擎

- 文本情绪分析（支持 10+ 种情绪）
- 数字人表情联动
- 情绪历史记录
- 玻璃态UI实时显示情绪状态

### 6. 数学公式支持

- 手写公式识别（SimpleTex API）
- KaTeX 实时渲染数学公式
- 支持 LaTeX 语法输入

## API 端点

### 教育系统 API (`/api/education`)

| 端点 | 方法 | 说明 |
|------|------|------|
| `/growth` | GET/POST | 获取/保存成长数据 |
| `/growth/add_exp` | POST | 增加经验值 |
| `/achievements` | GET | 获取成就列表 |
| `/achievements/unlock` | POST | 解锁成就 |
| `/collaboration` | GET/POST | 获取/保存协作记录 |
| `/collaboration/start_session` | POST | 开始协作会话 |
| `/collaboration/add_contribution` | POST | 添加贡献记录 |
| `/collaboration/end_session` | POST | 结束协作会话 |
| `/emotion/record` | POST | 记录情绪状态 |
| `/skills/record_usage` | POST | 记录技能使用 |

### 魔珐星云数字人 API (`/api/xingyun_digital_human`)

| 端点 | 方法 | 说明 |
|------|------|------|
| `/config` | GET | 获取数字人配置 |
| `/create_session` | POST | 创建数字人会话 |
| `/close_session` | POST | 关闭数字人会话 |
| `/start_session` | POST | 开始数字人会话 |
| `/drive_text` | POST | 文本驱动数字人 |
| `/drive_audio` | POST | 音频驱动数字人 |
| `/drive_emotion` | POST | 情绪驱动数字人 |
| `/active_sessions` | GET | 获取活跃会话列表 |
| `/health` | GET | 健康检查 |

### VRM 控制端点（备用）

| 端点 | 方法 | 说明 |
|------|------|------|
| `/vrm_emotion` | POST | 设置 VRM 表情（本地模式） |
| `/vrm_config` | GET | 获取 VRM 配置 |

## 使用方法

### 1. 启动服务

```bash
python server.py
```

服务将在 `http://localhost:3456` 启动

### 2. 访问界面

打开浏览器访问 `http://localhost:3456`，点击左侧菜单「教育数字人」

### 3. 配置魔珐星云数字人

在数字人配置面板输入：
- **App ID**: 魔珐星云应用ID
- **App Secret**: 魔珐星云应用密钥

系统会自动连接到 `https://nebula-agent.xingyun3d.com/user/v1/ttsa/session`

### 4. 使用教育功能

1. 选择一个教育技能模式
2. 开始对话，系统会自动记录协作过程
3. 查看成长面板了解进度
4. 使用数学助手时，可上传手写公式图片识别

### 5. 数学公式使用

- 在数学助手模式下，点击上传按钮上传手写公式图片
- 系统调用 SimpleTex API 识别并转换为 LaTeX
- 使用 KaTeX 实时渲染显示公式
