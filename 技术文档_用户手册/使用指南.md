# 教育数字人系统 - 使用指南

## 快速启动

### 方式一：双击启动脚本（推荐）
双击 `启动教育数字人_HTTPS.bat` 文件，系统会自动启动服务并打开独立界面。
- 支持 HTTPS（魔珐星云 SDK 需要）
- 敏感配置通过环境变量加载

### 方式二：手动启动
```bash
# 1. 设置环境变量（Windows PowerShell）
$env:XINGYUN_APP_ID = "your_app_id"
$env:XINGYUN_APP_SECRET = "your_app_secret"

# 2. 启动后端服务
py server.py --host 127.0.0.1 --port 3456

# 3. 打开浏览器访问
# 独立界面: https://127.0.0.1:3456/education-digital-human.html
```

## 访问地址

| 界面 | 地址 | 说明 |
|------|------|------|
| **独立界面** | https://127.0.0.1:3456/education-digital-human.html | 专为教育数字人设计的独立界面 |
| 主界面 | https://127.0.0.1:3456 | 完整的 super-agent-party 界面 |

> **注意**: 首次访问 HTTPS 会提示证书不受信任，点击"高级"→"继续访问"即可。

## 魔珐星云配置

敏感配置通过环境变量传入，避免硬编码：

| 环境变量 | 说明 |
|----------|------|
| `XINGYUN_APP_ID` | 魔珐星云应用 ID |
| `XINGYUN_APP_SECRET` | 魔珐星云应用密钥 |
| `SSL_KEYFILE` | SSL 私钥路径（可选） |
| `SSL_CERTFILE` | SSL 证书路径（可选） |

获取方式：
1. 登录 [魔珐星云平台](https://xingyun3d.com/)
2. 在应用中心创建驱动应用
3. 获取 appId 和 appSecret

## 常见问题

### Q: 浏览器提示"您的连接不是私密连接"？
A: 这是自签名证书的正常提示，点击"高级"→"继续访问"即可。

### Q: 魔珐星云连接失败？
A: 
1. 确保使用 HTTPS 访问（魔珐 SDK 不支持 HTTP + IP）
2. 检查环境变量是否正确设置
3. 确认 appId 和 appSecret 有效

### Q: VRM 数字人不显示？
A: 确保浏览器支持 WebGL，或尝试刷新页面

### Q: 对话无响应？
A: 检查后端服务是否正常运行，查看控制台日志
