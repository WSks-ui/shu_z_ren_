# 学术写作规范指南

## 一、论文基本结构

### 1.1 标题写作
**原则：简洁、准确、信息丰富**

好的标题应该：
- 准确反映研究内容
- 包含核心关键词
- 长度适中（通常不超过 20 个词）
- 避免缩写和行话

**示例对比：**
- ❌ 差：关于机器学习的一些研究
- ✅ 好：基于深度学习的个性化学习推荐系统研究

### 1.2 摘要撰写

**结构化摘要四要素：**
1. **背景/目的**：研究背景和目标
2. **方法**：研究设计和程序
3. **结果**：主要发现
4. **结论**：研究意义

**字数控制：**
- 中文摘要：300-500 字
- 英文摘要：150-300 词

### 1.3 引言写作

**倒三角结构：**
```
研究领域背景
    ↓
已有研究综述
    ↓
研究空白/问题
    ↓
研究目的和意义
```

**要点：**
- 说明研究的必要性
- 引用权威文献
- 明确研究问题
- 界定研究范围

### 1.4 方法部分

**必须包含：**
- 研究设计类型
- 研究对象/材料描述
- 数据收集程序
- 数据分析方法
- 伦理声明（如适用）

**写作要点：**
- 足够详细以供重复
- 说明选择的理由
- 报告样本量和功效

### 1.5 结果呈现

**数据呈现原则：**
- 客观报告发现
- 不做解释和评论
- 图表辅助说明
- 统计结果规范

**统计报告格式：**
```
t(df) = 值, p < .05, d = 效应量
F(df1, df2) = 值, p < .05, η² = 效应量
r(N) = 值, p < .05
```

### 1.6 讨论撰写

**结构建议：**
1. 主要发现总结
2. 与已有研究对比
3. 理论/实践意义
4. 研究局限性
5. 未来研究方向

## 二、学术语言规范

### 2.1 语言特点

**学术写作特征：**
- 客观性：避免主观情绪
- 准确性：精确使用术语
- 逻辑性：论证清晰连贯
- 正式性：避免口语表达

### 2.2 常见问题

**避免的表达：**
- ❌ "我觉得..." → ✅ "研究表明..."
- ❌ "非常好" → ✅ "显著改善"
- ❌ "很多" → ✅ "大量研究显示"
- ❌ "显然" → ✅ "这一结果表明"

**学术表达示例：**
| 口语表达 | 学术表达 |
|---------|---------|
| 看了文献 | 文献回顾显示 |
| 差不多 | 大致相当 |
| 非常重要 | 至关重要 |
| 这个说法 | 这一观点 |

### 2.3 段落结构

**主题句结构：**
```
主题句（论点）
    ↓
支持句1（证据/解释）
支持句2（证据/解释）
支持句3（证据/解释）
    ↓
总结句/过渡句
```

## 三、引用规范

### 3.1 引用原则

**何时引用：**
- 引用他人观点
- 使用他人数据
- 借鉴研究方法
- 支持自己论点

**何时不需引用：**
- 公共常识
- 自己原创观点
- 自己的研究数据

### 3.2 APA 7th 格式

**文内引用：**
```
单作者：(张三, 2024)
双作者：(张三, 李四, 2024)
三作者以上：(张三等, 2024)
```

**参考文献格式：**
```
期刊论文：
张三, 李四. (2024). 论文标题. 期刊名, 10(2), 100-120.

书籍：
王五. (2023). 书名. 出版社.

网络资源：
作者. (日期). 标题. 网站名. URL
```

### 3.3 GB/T 7714-2015 格式

**参考文献格式：**
```
[1] 作者. 文章标题[J]. 期刊名, 2024, 10(2): 100-120.
[2] 作者. 书名[M]. 出版地: 出版社, 2023: 50-60.
[3] 作者. 文章标题[EB/OL]. (发布日期)[引用日期]. URL.
```

## 四、学术诚信

### 4.1 学术不端行为

**主要类型：**
1. **抄袭**：未注明来源使用他人成果
2. **伪造**：编造或篡改数据
3. **代写**：他人代替完成学术工作
4. **重复发表**：同一成果多次发表
5. **不当署名**：署名与贡献不符

### 4.2 AI 使用规范

**合理使用：**
- 辅助文献检索
- 语言润色建议
- 格式检查

**需要声明：**
- 在方法或致谢中说明
- 明确 AI 辅助的范围
- 保证内容的原创性

**声明示例：**
> 本研究在写作过程中使用了 [AI工具名称] 进行语言润色，所有学术内容由作者独立完成。

### 4.3 自我核查清单

- [ ] 所有引用都有出处
- [ ] 引用格式统一规范
- [ ] 数据真实可靠
- [ ] 图表标注正确
- [ ] 无抄袭内容
- [ ] 署名合理
- [ ] 已做 AI 声明（如适用）

## 五、投稿指南

### 5.1 期刊选择

**考虑因素：**
- 期刊范围与匹配度
- 影响因子和声誉
- 审稿周期
- 开放获取选项
- 版面费用

### 5.2 投稿材料

**通常需要：**
- 投稿信（Cover Letter）
- 论文正文
- 图表文件
- 利益冲突声明
- 作者贡献声明
- 伦理审批文件（如适用）

### 5.3 审稿流程

```
投稿 → 编辑初审 → 送审 → 审稿意见 → 修改 → 录用/拒稿
```

**修改要点：**
- 逐条回复审稿意见
- 修改处标记清楚
- 态度诚恳专业
- 有理有据解释

## 六、常见错误与纠正

### 6.1 结构问题

| 错误 | 纠正 |
|-----|------|
| 引言过长 | 控制在全文10-15% |
| 方法不详 | 提供足够细节 |
| 结果解释 | 放到讨论部分 |
| 结论重复 | 总结而非重复 |

### 6.2 语言问题

| 问题 | 示例 | 纠正 |
|-----|------|------|
| 句子过长 | 超过3行的句子 | 拆分为短句 |
| 用词不当 | "证明" | "表明/显示" |
| 主观表述 | "我认为" | "研究表明" |
| 逻辑不清 | 跳跃论证 | 添加过渡 |

### 6.3 引用问题

| 问题 | 纠正方法 |
|-----|---------|
| 引用过少 | 补充相关文献 |
| 引用过时 | 更新最新研究 |
| 二手引用 | 查阅原文 |
| 格式混乱 | 统一规范 |
