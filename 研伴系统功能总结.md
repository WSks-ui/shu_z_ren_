# 研伴系统功能总结

## 一、教育技能模块

| 技能 | 功能描述 | 核心能力 |
|------|----------|----------|
| **科研助手** | 苏格拉底式引导实验设计 | 选题探索→假设构建→实验设计→数据分析阶段推进 |
| **文献综述** | PRISMA系统综述方法指导 | PICO框架设计、检索策略、质量评估、综述撰写 |
| **论文写作** | 学术论文写作协助 | 结构规划、语言润色、引用格式（APA/GB/T 7714/IEEE） |
| **虚拟导师** | 个性化学习支持 | 知识讲解、学习规划、答疑解惑、类比教学 |
| **数学助手** | 数学问题求解与公式识别 | 公式识别转LaTeX、分步解题、错题分析 |

---

## 二、成长系统

| 功能 | 说明 |
|------|------|
| **经验值系统** | 每次对话获得经验值，量化学习投入 |
| **等级系统** | 每100经验升一级，反映能力进阶 |
| **成就系统** | 8种成就徽章（初次对话、文献读者、实验设计师等） |
| **学习统计** | 对话次数、论文阅读、实验设计、学习时长等数据追踪 |

---

## 三、协作记录系统

| 功能 | 说明 |
|------|------|
| **会话记录** | 按类型分类存储（科研/文献/论文/辅导） |
| **贡献统计** | AI与用户贡献比例可视化 |
| **协作摘要** | 自动生成会话摘要 |
| **记录导出** | 支持Word格式导出协作记录 |

---

## 四、知识库系统

| 知识库 | 内容 |
|--------|------|
| **教育研究方法** | 研究设计、定量/定性方法、混合研究 |
| **学术写作规范** | 论文结构、语言规范、引用格式 |
| **跨学科研究方法** | 跨学科研究框架与方法 |
| **AI局限性与学术伦理** | AI边界、人机协作原则、学术诚信 |

---

## 五、数字人与交互

| 功能 | 说明 |
|------|------|
| **魔珐星云数字人** | 云端渲染3D数字人，支持语音播报 |
| **情绪映射** | 识别10+种情绪，联动数字人表情 |
| **语音识别** | 支持语音输入（需配置ASR模型） |
| **语音合成** | 多引擎TTS，支持数字人播报 |
| **流式响应** | 实时输出，降低首字节延迟 |

---

## 六、对话系统

| 功能 | 说明 |
|------|------|
| **多技能切换** | 五种教育技能自由选择 |
| **阶段进度条** | 可视化显示当前阶段，支持跳转 |
| **历史记录** | 对话历史保存与恢复 |
| **上下文记忆** | 记住上次讨论内容，支持继续 |
| **场景模板** | 预设问题模板，快速开始对话 |
| **文件上传** | 支持图片、PDF等文件解析 |

---

## 七、导出与报告

| 功能 | 说明 |
|------|------|
| **协作记录导出** | 单个/批量导出Word文档 |
| **学习报告生成** | 汇总成长数据、协作统计、技能使用、成就 |
| **报告导出** | 学习报告Word格式导出 |

---

## 八、系统设置

| 功能 | 说明 |
|------|------|
| **多模型支持** | 支持OpenAI、阿里云、DeepSeek等多LLM后端 |
| **快速模型配置** | 可配置快速模型用于简单对话 |
| **思考模式** | 支持启用模型思考过程输出 |
| **主题切换** | 亮色/暗色主题 |

---

## 九、界面功能

| 功能 | 说明 |
|------|------|
| **沉浸式布局** | 数字人全屏背景，浮动功能面板 |
| **快捷操作栏** | 技能、成长、记录、书签快速访问 |
| **知识书签** | 收藏重要对话内容 |
| **响应式设计** | 适配不同屏幕尺寸 |

## API 接口列表

### 教育系统 API (`/api/education`)

| 端点 | 方法 | 说明 |
|------|------|------|
| `/growth` | GET/POST | 获取/保存成长数据 |
| `/growth/add_exp` | POST | 增加经验值 |
| `/achievements` | GET | 获取成就列表 |
| `/achievements/unlock` | POST | 解锁成就 |
| `/collaboration` | GET/POST | 获取/保存协作记录 |
| `/collaboration/start_session` | POST | 开始协作会话 |
| `/collaboration/add_contribution` | POST | 添加贡献记录 |
| `/collaboration/end_session` | POST | 结束协作会话 |
| `/collaboration/stats` | GET | 获取协作统计 |
| `/collaboration/export` | POST | 导出协作记录 |
| `/chat` | POST | 教育对话接口 |
| `/chat/stream` | POST | 流式对话接口 |
| `/chat/history` | GET | 获取对话历史 |
| `/emotion/record` | POST | 记录情绪状态 |
| `/skills/prompts` | GET | 获取技能提示词 |
| `/knowledge/search` | POST | 搜索知识库 |
| `/report/generate` | GET | 生成学习报告 |
| `/report/export` | POST | 导出学习报告 |
