# 教育数字人演示文稿

## 快速启动

### Windows
双击运行 `start-edu-demo.bat`

### 手动启动
```bash
cd web
npm install
npx vite edu-index.html --port 5174
```

然后访问 http://localhost:5174

## 演讲要点

### 01 开场
- 「研伴——一个面向高校教育场景的智能科研协作数字人」
- 「它不是简单的问答工具，而是能够与学习者共同探索、协同创造的学术伙伴」

### 02 数字人
- 云端渲染，无需本地 GPU
- 低带宽可用，随时随地访问
- 情绪联动，有温度的交互

### 03 教育技能
- 苏格拉底式引导，培养独立思考
- 五大技能覆盖科研全流程
- 支持图片识别、番茄钟、笔记生成

### 04 成长系统
- 经验值与等级系统
- 12 枚成就徽章，分四类
- 学习进步有据可查

### 05 人机协作
- 记录每次协作，见证共同成长
- 人机贡献比例可视化
- 学习报告导出

### 06 技术架构
- Vue.js 3 前端 + FastAPI 后端
- RAG 向量检索知识库
- 学术 API 集成（arXiv、Semantic Scholar）

### 07 结束
- 「研伴——协作者，而非答案生成器」
